💡 薬機法簡単ガイド

🎯 このガイドの目的

美容・健康商品の広告制作者が、薬機法（医薬品医療機器等法）の基本を理解し、適切な表現で魅力的な広告を作成できるよう支援します。

🔍 薬機法とは？

薬機法（医薬品医療機器等法）は、化粧品や医薬品、医療機器の品質・有効性・安全性を確保するための法律です。美容系商品の広告では、効果効能の表現に厳格な制限があります。

なぜ規制があるの？

  • 消費者を誤解や健康被害から守るため
  • 不当な競争を防ぎ、市場の公正性を保つため
  • 医薬品と化粧品の区別を明確にするため

📊 商品カテゴリ別の表現ルール

🧴 化粧品（一般化粧品）

⚠️ 表現できないこと：
  • 医学的・薬理的な効果効能
  • 「美白」「シワ改善」等の医薬部外品専用表現
  • 疾患名や症状の改善を示唆する表現
✅ 適切な表現例：
・うるおいを与える
・乾燥から肌を守る
・肌をなめらかに整える

💊 薬用化粧品・医薬部外品

📋 表現できること：
  • 承認された効能効果の範囲内
  • 「美白」「シワ改善」等（有効成分による）
  • 予防的な表現
✅ 適切な表現例：
・メラニンの生成を抑え、シミ・そばかすを防ぐ
・肌荒れを防ぐ
・乾燥による小ジワを目立たなくする

🌿 サプリメント・健康食品

⚠️ 特に注意すべきこと：
  • 疾病の診断・治療・予防効果の標榜禁止
  • 身体の組織機能の増強・増進表現の制限
  • 医薬品的な効能効果の表現禁止
✅ 適切な表現例：
・健康的な毎日をサポート
・栄養バランスを整える
・美容に配慮した成分配合

🚫 絶対に避けるべきNG表現

NG表現 理由 適切な表現例
「シミが消える」 医薬品的効果の標榜 「透明感のある肌へ」
「アンチエイジング」 老化防止は医薬品的表現 「年齢に応じたケア」
「細胞を活性化」 身体機能への作用 「肌にハリを与える」
「ダイエット効果」 痩身効果は医薬品的 「スタイルケアをサポート」

✍️ 表現テクニック

🎨 魅力的で適切な表現のコツ

  • 感覚的な表現を活用：「しっとり」「さらさら」「ふっくら」
  • 期待感を演出：「〜へと導く」「〜をサポート」
  • 使用感に焦点：「心地よい使用感」「リラックスタイム」
  • イメージ訴求：「輝くような印象」「健やかな毎日」

🔄 NG表現の言い換えテクニック

Before: 「肌の再生を促進」
After: 「肌のキメを整える」
Before: 「血行を改善」
After: 「肌にうるおいとハリを与える」

🎯 文章タイプ別のポイント

📢 キャッチコピー

短く印象的でありながら、薬機法に準拠した表現を心がける。誇大表現は特に注意。

📰 LP見出し・タイトル

読者の興味を引きつつ、医薬品的効果を連想させない表現にする。

📝 商品説明文

商品の特長や使用感を詳しく説明できるが、効能効果の範囲は遵守する。

💬 お客様の声

体験談でも薬機法の制約は同様。個人の感想として適切な範囲内で表現。
